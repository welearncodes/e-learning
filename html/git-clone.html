Git Clone

Now that you have made your own repository and have acuired mad skills, you might want to copy someone elses repository

You can do this by going to the location on your computer where you want to add the files. Then you go to https://github.com and find a project you wish to clone and copy that projects link.

If you want to copy this website you would write:

git clone https://github.com/welearncodes/e-learning.git

This will copy the files to the location you picked out.
